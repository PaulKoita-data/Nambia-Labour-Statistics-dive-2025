# Namibia Economic & Labour Market Dashboard

## Overview
This project analyzes Namibia’s labour market and macroeconomic conditions using official statistical data. 
The analysis integrates employment, inflation, youth outcomes, and income distribution into a single analytical workflow using PostgreSQL and Power BI.

## Key Indicators
- Employment-to-population ratio
- National unemployment rate
- Youth NEET rate
- Inflation trends (CPI)
- Employment and youth employment by industry
- Income distribution snapshot (2023)

## Tools & Technologies

- PostgreSQL – Relational database used for structured storage, schema design, and analytical SQL queries.
- Python – Data cleaning and transformation for selected datasets prior to database loading.
- Microsoft Excel – Initial data inspection, date-format standardization, and preprocessing to ensure compatibility across tools.
- Power BI– Interactive dashboard development, KPI creation, and data visualization.
